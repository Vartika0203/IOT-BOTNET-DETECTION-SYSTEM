# 🔐 IoT Botnet Detection System

Real-time Network Intrusion Detection System using Machine Learning (Random Forest)

## 📊 Features

- **Live Traffic Visualization** - Color-coded graph (Green = Normal, Yellow = Suspicious, Red = Attack)
- **Real-time Attack Detection** - Identifies DDoS, Port Scan, Reconnaissance, C&C Communication
- **Explainable AI** - Shows feature importance for each detection
- **Confidence Scoring** - Percentage confidence for every prediction
- **Adaptive Baseline** - Learns normal traffic patterns
- **Attack Tracking** - Maps top attacking IP addresses
- **Model Performance Metrics** - Precision, Recall, F1-Score display

## 🎯 Performance

|--------|-------|
| Precision | 94% |
| Recall | 91% |
